# Prospector

Automated job hunting at scale, powered by GitHub Actions.

## Tools

- **Scraper** - Aggregate job postings from multiple sources
- **Tailor** - Customize resumes for specific job descriptions
- **Submitter** - Automate application submissions

## Setup

1. Fork this repo
2. Configure secrets in repository settings
3. Enable GitHub Actions workflows

## Workflows

All tools run as GitHub Actions workflows on schedule or manual trigger.
